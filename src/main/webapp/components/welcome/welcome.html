Welcome to TrunkDB

TrunkDB is an online relational database management system for researchers and analysts. You can work on databases as a team across institutions, import data from Microsoft Access and text files, and publish query results as datasets and embed them in websites and dashboards.

TrunkDB is hosted on public cloud infrastructure and supported by Cetis LLP. The code that runs this service is open source and licensed under the Apache Software License.

TrunkDB started life as the Online Research Database Service (ORDS) at the University of Oxford, and was developed with the support of Jisc.

We're in private beta right now, but if you'd like to get involved, drop an email to info@cetis.org.uk
